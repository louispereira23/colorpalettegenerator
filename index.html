About Chromaword

Chromaword is a minimalist tool that generates harmonious color palettes and font pairings based on words. Perfect for kickstarting your web design projects with cohesive, meaningful color schemes.

Simply enter a word, and Chromaword will create a complete web design palette with:

  • Background color - Light and subtle for content readability
  • Text color - With proper contrast for accessibility
  • Accent color - For buttons, links, and highlights
  • Accent text color - For text on accent backgrounds
  • Complementary font pairings - Heading and body fonts that work together

Use the refresh button to generate alternative palettes for the same word, or try different words to explore various design options.

The color generation algorithm creates consistent, deterministic results based on the input word, ensuring that the same word always produces the same palette.

Export CSS Variables

Copy these CSS variables to use in your project:

:root {
  --background: #ffffff;
  --text: #333333;
  --accent: #4A6FFF;
  --accent-text: #ffffff;
  --heading-font: 'Playfair Display', serif;
  --body-font: 'Inter', sans-serif;
}
